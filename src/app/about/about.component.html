NEST is a full stack application designed collaboratively by Ingeborg Slegers, Jessica Brandt, and Robert Baxter. This app includes an Angular 7 front end that interfaces with a back end web API designed with Node.js, Express, and PostgreSQL. The both github repositories are accessible from the github icon on the bottom right of the footer.
